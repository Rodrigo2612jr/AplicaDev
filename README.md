# AplicaDev

Landing page em React + Vite para uma empresa SaaS que desenvolve sistemas, aplicativos e sites.

## Stack

- React
- TypeScript
- Vite
- CSS customizado

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Observacao

No estado atual, os CTAs da home apontam para a secao interna de diagnostico em `src/App.tsx` na constante `diagnosticHref`.
Quando a pagina real do formulario estiver pronta, basta substituir esse destino pela rota ou URL definitiva.

A identidade visual da LP foi atualizada para a paleta da marca (tons de cinza e verde) e usa assets locais em `src/assets`.
Foi criada uma versao vetorial transparente da marca para uso imediato; quando voce enviar o arquivo oficial em alta resolucao (SVG/PNG), a troca sera direta.
